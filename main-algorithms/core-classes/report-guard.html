The ReportGuard class

Objects of this type can be used to enable printing of some information during various of the computation in libsemigroups_pybind11. Reporting is enabled (or not) at construction time, and disable when the ReportGuard goes out of scope.

Full API

class ReportGuard
__init__(self: ReportGuard, val: bool = True) → None

Constructs a ReportGuard with reporting enabled by default.

Parameters:

val (bool) – whether to report or not (default: True).

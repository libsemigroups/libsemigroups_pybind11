The LeftAction class

Class representing a left action of a semigroup or monoid on a set.

This page contains the documentation for the class LeftAction, which just calls Action with the keyword arguments func given by ImageLeftAction and side given by side.left.

Full API

class LeftAction

Bases: Action

__init__(self: LeftAction, generators: list[Element] = None, seeds: list[Point] = None) → None

Construct an Action from generators and seeds, ImageLeftAction and side.left.

Keyword Arguments:

  • generators (list[Element])– at least one generator for the action.

  • seeds (list[Point]) – at least one seed point for the action.

Raises:

  • TypeError – if generators or seeds is not a list.

  • ValueError – if generators or seeds has length 0.

  • KeyError – if the action given by the arguments is not yet implemented.

Important

The class LeftAction has all the methods of the Runner class but, for boring technical reasons, is not a subclass of Runner. If thing is an instance of LeftAction, then you can use thing as if it were an instance of Runner but isinstance(thing, Runner) will return False.

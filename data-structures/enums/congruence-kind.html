The congruence_kind enum

This page describes the enum class congruence_kind in libsemigroups_pybind11 for representing the whether a congruence is one- or two-sided.

Full API

class congruence_kind

The values in this class can be used to indicate that a congruence should be 1-sided or 2-sided.

Members:

onesided : Type for 1-sided congruences (right).

twosided : Type for 2-sided congruences.

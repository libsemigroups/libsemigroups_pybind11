The tril enum

This page describes the enum class tril in libsemigroups_pybind11 for representing values that can either be true, false, or not currently known.

Full API

class tril

The values in this enum can be used to indicate a result is true, false, or not currently known.

The valid values are:

false = <tril.false: 0>

Value representing false.

true = <tril.true: 1>

Value representing true.

unknown = <tril.unknown: 2>

Value representing unknown (either true or false).
